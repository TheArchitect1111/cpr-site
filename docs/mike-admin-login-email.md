Subject: CPR Admin Login — what changed and how to sign in today

Hi Mike,

We fixed the admin login issue on CPR Global. The password field had been removed from the sign-in page while recovery instructions still pointed you there — that’s resolved now. Here’s how to get in.

══════════════════════════════════════════
RECOMMENDED: EMAIL LOGIN LINK (NO PASSWORD)
══════════════════════════════════════════

This is the fastest way in. No password to remember.

1. Go to: https://canadianprospectrecruitment.vercel.app/admin/login
2. Scroll to **Email me a login link**
3. Enter: **mikecprglobal@mississaugamagic.com**
4. Click the button and check your inbox (and spam folder)
5. Open the email and tap **Sign in to CPR Admin** within 15 minutes

You’ll land in the Admin Console. That same login also unlocks portal owner tools (Update Hub, family portal preview, etc.).

══════════════════════════════════════════
ALTERNATIVE: PASSWORD SIGN-IN
══════════════════════════════════════════

The sign-in page now has a **Sign in with password** section again at the top.

Use this if you have a password on file (temporary password we set, or one you chose after a reset).

1. Go to: https://canadianprospectrecruitment.vercel.app/admin/login
2. Enter your email and password
3. Click **Sign in**

If you don’t know your password, use the email login link above instead — it always works when email is configured.

══════════════════════════════════════════
FORGOT PASSWORD
══════════════════════════════════════════

1. On the sign-in page, click **Forgot password?**
2. Enter your admin email
3. Check your inbox for the reset link (expires in 30 minutes)
4. Choose a new password, then sign in

If you see a message that password reset isn’t fully configured, use **Email me a login link** on the sign-in page — you don’t need a password for that path.

══════════════════════════════════════════
QUICK LINKS AFTER YOU’RE IN
══════════════════════════════════════════

• Admin Console: https://canadianprospectrecruitment.vercel.app/admin
• Family portal (owner view): https://canadianprospectrecruitment.vercel.app/portal/owner
• System Readiness (check configuration): open Admin Console → System Readiness section
